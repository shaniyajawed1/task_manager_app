## 📋 Project Overview

**TaskFlow** is a full-featured task management system built for the **Anotech India Solutions SDE-1 Assignment**. It helps teams and individuals organize, track, and manage their daily tasks efficiently.

> "Not just forms and tables — a productivity tool someone would actually use."

### 🎯 What Problem Does It Solve?

- ❌ Tired of forgetting important tasks?
- ❌ Struggling to track task progress?
- ❌ Need a simple way to manage team work?

**TaskFlow solves all of this** with an intuitive, beautiful interface.
